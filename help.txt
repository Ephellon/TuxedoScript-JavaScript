/********
  Tuxedo Script Help:
    * general rules/notes: the Tuxedo escape characters aren't needed in them, i.e. \#
    - all loops/function need a matching semi-colon
    - place comments on the NEXT line, or the code may become useless
    - when changing a loop with eval enabled, delete the entire loop first, put an error before hand
    - place dot assignments on a newline, i.e.:
    $doc.$id(id).$html = 'foo-bar'.
    #e = "hi".
      // ugly code
      // instead, use:
    $doc.$id(id).$html
      = 'foo-bar'.
    #e = "hi".

--------
HTML attributes: attribute - meaning
    type - [input] required, can be any of the following [ts, tux, tuxedo, tuxedoscript, tuxedo-script, tscript, t-script]
    type - [output] optional, can be one of the following [tso, tuxo, tuxedoo, tuxedoscriptout, tuxedo-script-out, tscriptout, t-script-out]
    eval=! - execute the code
    editor=! - try to style the code for easier reading
    clean=! - remove the header and footer of the code
    math=! - enable shorthand math, i.e. 2^5 = Math.pow(2, 5)
    advance=! - enable advanced features
    ugly=! - shorten code
    hide=! - hide this element
    legacy=! - switch newer features to old versions
    wordy=! - allow use of "normal" English words

Experimental Features: experiment - meaning

    notch - the no.t.ch (No Tuxedo Channel)
      disable ALL Tuxedo execution within ##/ ... /##
    sotch - the so.t.ch (Some Tuxedo Channel, or Shortcut Table Channel)
      disable some Tuxedo execution within ##- ... -## [go straight to shortcuts]
    totch - the to.t.ch (Total Tuxedo Channel)
      enable all Tuxedo execution within ##\ ... \## [or simply disable notch and sotch]
